Status: Source of Truth (SSOT)

## System Definition

The NSQ Ecosystem is a human-governed, tower-based agentic operating system composed of:

- ACI — Agentics Command Interface
- GCOL — Generative & Coordination Operational Layer
- UXL — User Experience Layer

Governed by:
- HGC³AE² Framework
- Skipjack Protocol

---

## 1. Three-Layer Architecture

### ACI (Command Layer)
- Morning Command Deck
- Persona-aware interface
- Command + Execute flow
- Approvals + Handoffs
- Entry into Operations View

### GCOL (Control Plane)
- Tower-based coordination model
- Governed routing between domains
- Context preservation
- Validation + checkpoints
- Escalation to human

### UXL (Experience Layer)
- Based on NSQ interface
- Unified entry point (no fragmentation)
- Content + Product delivery surfaces

---

## 2. Governance-First Doctrine (HGC³AE²)

- Human authority is absolute
- Governance defines execution
- Context is curated continuously
- Collaboration is ongoing
- Execution must match design + runtime

---

## 3. Skipjack Protocol (Enforced)

System MUST:
- Perform live validation
- Prevent silent fallback
- Signal uncertainty explicitly
- Enforce reconciliation checkpoints
- Gate execution through governance
- Verify runtime assumptions

HARD RULE: NO SILENT CONTINUATION
